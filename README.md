# ToDo Manager - Progetto Basi di Dati e Object Oriented

Sistema gestionale per task personali e collaborativi.

## 📂 Struttura della Repository
- **`/Database`**: Contiene lo script SQL per la creazione dello schema e la documentazione.
- **`/Object_Oriented`**: Codice sorgente e documentazione dell'applicativo.

## 🛠️ Requisiti e Avvio
1. Configurare il database PostgreSQL tramite lo script in `/database`.
2. Importare la cartella `Object_Oriented` in IntelliJ IDEA.
